# Kubernetes

#### Installation

```
Option 1
$ brew install kubernetes-cli

Option 2
$ gcloud components install kubectl

Verification
$ kubectl version
```

#### Cheatsheet

```
$ kubectl cluster-info    # URL response means kubectl is correctly configured to access your cluster.
```

#### Minikube

```
$ brew cask install minikube
$ minikube version
$ minikube start
```
